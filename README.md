

Notes Manager
Система управління нотатками, побудована на базі асинхронного сервера та сучасних методів розробки API. Проект забезпечує стабільну роботу з текстовими даними, високу швидкість обробки запитів та чіткий розподіл логіки.

Команда проекту
Scrum-менеджер: [Ваше Ім'я] — координація процесів розробки, управління беклогом та документацією.

Developer: [Ім'я девелопера] — розробка серверної архітектури та реалізація програмної логіки.

Tester: [Ім'я тестера] — забезпечення якості продукту, розробка та виконання тестових сценаріїв.

Технологічний стек
Мова програмування: Python.

Framework: Easy API — для побудови інтерфейсу програмування.

Сервер: Uvicorn — ASGI сервер для запуску додатка.

Функціональні вимоги
Створення та збереження нотаток із метаданими.

Отримання повного списку записів через API.

Пошук та фільтрація нотаток за ключовими словами.

Редагування та видалення існуючих даних.

Інструкція із розгортання
Налаштування середовища
Для роботи проекту необхідно клонувати репозиторій та встановити залежності:

Bash
git clone https://github.com/igorkrasher/note-master.git
cd note-master
pip install easy-api uvicorn
Запуск додатка
Запуск сервера для обробки вхідних запитів:

Bash
uvicorn src.main:app --reload
Структура репозиторію
src/ — вихідний код системи (моделі даних, маршрути API).

tests/ — набір тестів для перевірки цілісності системи.

docs/ — технічна документація та проектна специфікація.
